DEMO

Fantastic! Let's set up your authenticator application.

This will secure your account by providing a unique, temporary code that we'll periodically ask for when you access information. This helps us know that you're you! You'll be able to find this temporary code within the app after scanning the QR code provided below. If you're unsure of which authenticator application to use for this registration, here are a few that you can choose from:

  • Authy
  • Google Authenticator
  • Microsoft Authenticator

Once you have the application installed on your phone, open it up, and "add" an account by scanning this QR code. If all goes well, it'll provide you with a code that changes every 30 seconds or so. We'll enter this code in the next step.

What's Next
